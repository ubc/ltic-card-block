# WP Card Block

A flexible WordPress block plugin that provides various card layouts for your content.

## Description

The **WP Card Block** plugin introduces a versatile card block to the Gutenberg editor. It supports multiple layout variations, custom styles, and integrates seamlessly with Query Loops for dynamic content display.

## Features

-   **Multiple Layout Variations**:
    -   Image + Text + Button
    -   Image + Text
    -   Text + Button
-   **Custom Styles**:
    -   Default
    -   CMS Card
    -   CTLT Card
-   **Query Loop Integration**: Automatic support for dynamic data when used inside a Query Loop.
-   **Responsive Design**: Built with responsive layouts in mind.

## Installation

1.  Clone this repository into your WordPress plugins directory:
    ```bash
    cd wp-content/plugins
    git clone <repository-url> wp-card-block
    ```
2.  Activate the plugin through the WordPress admin dashboard (Plugins > Installed Plugins).

## Development

To start development mode with file watching:

```bash
npm install
```

```bash
npm start
```
